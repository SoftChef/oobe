# OOBE

優質的表單操作庫，你開心也可以用在後端環境上。

[![NPM Version][npm-image]][npm-url]

![flow][Flow]

## Install

```bash
npm i --save oobe
```

## How to use

```js
import OOBE from 'oobe'
let oobe = new OOBE()
```

### Container

模組化精靈組的物件，詳細請見[文件][DocLink]。
